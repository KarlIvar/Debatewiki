# DebateWiki
An encyplopedia of arguments.

DebateWiki aspire to be the first place you go when you want a multifaceted view of a topic. If you want to find the most common arguments

DebateWiki is a combination of an encyclopedia and a debate forum.

It is an encylopedia in the sense that the provides community-created knowledge detached from its users. Just like other encyclopedias, its purpose is to provide information and in an honest and unbiased manner.

It is a forum in the sense that it is encapulates discussion. If you think an argument is weak or invalid, you can publicly share a counterargument explaining why. If you agree with a belief or an argument and want to see how it is challanged, you can browse the most popular counterarguments. This process continues recursively, making a large web of arguments.
